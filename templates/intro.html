Project Overview

This project is a Sentiment Analysis Platform designed to analyze customer reviews and feedback using Natural Language Processing (NLP) techniques. Built using Flask, the platform allows users to upload datasets containing reviews, which are then processed and analyzed to classify reviews into Positive, Negative, or Neutral sentiments. The system also provides insights into customer emotions, such as joy, anger, and calmness, based on the sentiment of the reviews. In addition to sentiment classification, the project integrates a machine learning component using a Random Forest Classifier to train on the user-uploaded dataset and provide sentiment predictions. The web interface is built with user-friendly features, including file upload, real-time sentiment analysis, and downloadable results.

Key Features

User-friendly Interface:

  • Intuitive and clean design for users to easily upload review datasets.
  • Results are displayed in a tabular format, showcasing sentiment and emotion for each review

Real-time Sentiment Analysis:

  • As soon as a dataset is uploaded, the system preprocesses the data and runs a sentiment analysis using the TextBlob library.
  • Sentiments (Positive, Negative, Neutral) are identified, and corresponding emotions are assigned based on the analysis.

Machine Learning Integration

  • The platform trains a Random Forest Classifier model on the uploaded dataset to enhance sentiment prediction capabilities.
  • Both the model and the vectorizer used for feature extraction are saved for future use, ensuring faster predictions for similar datasets.

    • Data Visualization

  • A bar chart representing the distribution of sentiments (Positive, Negative, Neutral) is generated and displayed on the dashboard.
  • Users can download this chart for further analysis and presentations.

    • Downloadable Results:

  • The analyzed dataset, including sentiment and emotion classifications, is available for download as a CSV file.
  • The sentiment distribution chart is also downloadable for users who want a visual summary of the analysis.

    • Data Preprocessing:

  • Text reviews are preprocessed to remove noise (punctuation, numbers, and stopwords).
  • Lemmatization is applied using spaCy to normalize words and improve the accuracy of the sentiment analysis.

    • Technologies Used

    1. Flask: A lightweight web framework used to build the web interface, handle file uploads, and serve the sentiment analysis results.
    2. Pandas: For data manipulation and handling CSV file uploads. It helps in organizing the review data and preparing it for analysis.
    3. TextBlob: Used for basic sentiment analysis to classify reviews as Positive, Negative, or Neutral.
    4. SpaCy: Employed for lemmatization during text preprocessing, ensuring accurate and meaningful analysis of the reviews.
    5. Random Forest Classifier (from Scikit-learn): Machine learning algorithm used for training a model on the user-uploaded review data, improving sentiment classification.
    6. Matplotlib: For generating the sentiment distribution bar chart that visualizes the breakdown of customer sentiment in the dataset.
    7. Bootstrap/CSS: Custom CSS and Bootstrap were used to ensure a responsive and visually appealing user interface.
    8. SQLite/SQLAlchemy (Optional): For storing user credentials securely in case a login system is implemented.
